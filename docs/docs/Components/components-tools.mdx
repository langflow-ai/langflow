The **Tools** category was removed from the **Components** menu in Langflow version 1.5.

You can find the components that previous belonged to this category in the following locations:

* The [**Calculator** component](/components-helpers#calculator) moved to the **Helpers** core components.
* The [**MCP Tools** component](/components-agents#mcp-connection) moved to the **Agents** core components.
* The [**Python Interpreter** component](/components-processing#python-interpreter) replaces the **Python REPL** and **Python Code Structured** components.
* All other components were moved to one of the following bundles:
    * [arXive bundle](/bundles-arxiv)
    * [Bing bundle](/bundles-bing)
    * [DataStax bundle](/bundles-datastax)
    * [DuckDuckGo bundle](/bundles-duckduckgo)
    * [Exa bundle](/bundles-exa)
    * [Glean bundle](/bundles-glean)
    * [Google bundle](/bundles-google)
    * [Icosa Computing bundle](/bundles-icosacomputing)
    * [LangChain bundle](/bundles-langchain)
    * [SearchApi bundle](/bundles-searchapi)
    * [Wikipedia bundle](/bundles-wikipedia)
