This page describes the components that are available in the **openGauss** bundle.

## openGauss vector store

The **openGauss** component reads and writes to [openGauss DataVec](https://docs.opengauss.org/zh/docs/latest/datavec/datavec_overview.html) vector stores using an instance of [`OpenGauss`](https://docs.langchain.com/oss/python/integrations/vectorstores/opengauss).

### openGauss vector store parameters

You can inspect a vector store component's parameters to learn more about the inputs it accepts, the features it supports, and how to configure it.

<PartialParams />

<PartialConditionalParams />

For information about accepted values and functionality, see the [openGauss DataVec documentation](https://docs.opengauss.org/zh/docs/latest/datavec/datavec_overview.html) or inspect [component code](/concepts-components#component-code).

| Name                 | Type         | Description                                                |
| -------------------- | ------------ | ---------------------------------------------------------- |
| connection_string    | SecretString | The openGauss Server Connection String.                    |
| table_name           | String       | The table name for the vector store.                       |
| search_query         | String       | The query for similarity search.                           |
| ingest_data          | JSON         | The data to be ingested into the vector store.             |
| embedding            | Embeddings   | The embedding function to use.                             |
| index_type           | String       | The vector index type (HNSW or IVFFLAT).                   |
| distance_strategy    | String       | The distance strategy (COSINE, EUCLIDEAN, INNER_PRODUCT, MANHATTAN). |
| embedding_dimension  | Integer      | The dimension of the embedding vectors.                    |
| number_of_results    | Integer      | The number of results to return in search.                 |
