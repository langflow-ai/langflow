Deploy Langflow on Render

This guide explains how to deploy Langflow on Render, a cloud platform for deploying web applications and APIs.

note

Langflow requires at least 2 GB of RAM to run, so it uses a standard Render instance. This may require a credit card. Review Render's pricing before proceeding.

  1. Click the following button to go to Render:

  2. Enter a blueprint name, and then select the branch for your render.yaml file.

  3. Click Deploy Blueprint.

Wait for the deployment to complete.

Your Langflow instance is now ready to use.
